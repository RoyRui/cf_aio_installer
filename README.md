# Cloud Foundry All In One Installer

## Overview

Cloud Foundry All In One Instalelr is a tool set for building Cloud Foundry v2 devboxes with BOSH and OpenStack.

Pre-Alpha version.

## Tools

* DevStack
* bosh-bootstrap
* bosh-cloudfoundry
* nginx as a reverse proxy

## Requirements

* Ubuntu 12.04 64bit
* 8GB+ Memory
* 50GB+ Disk Space

## Install

Run `bin/bootstrap.sh`.
